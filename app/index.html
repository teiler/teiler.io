Teiler.io Web Client

What is the difference to all these other split apps?

Teiler.io is a tool to split expenses in groups. It is implemented as a public REST API everyone can use. We developed one implementation for the web platform because we think it’s the most common one. But if you want a native app, then you can simply implement this API and you’re good to go.

Teiler.io is open-source. You can contribute to all the products we released. Have a look at our organization on GitHub!
